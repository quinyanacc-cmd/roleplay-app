# Roleplay Tagesreview v1.4

Diese Web-App speichert alle persönlichen Eingaben ausschließlich lokal im Browser des verwendeten Geräts.

## Veröffentlichung über GitHub Pages

1. Alle Dateien aus diesem Ordner in das Repository `roleplay-app` hochladen.
2. In GitHub `Settings` öffnen.
3. `Pages` öffnen.
4. `Deploy from a branch` auswählen.
5. Branch `main` und Ordner `/ (root)` auswählen.
6. Speichern.
7. Die angezeigte GitHub-Pages-Adresse in Safari öffnen.
8. Teilen → Zum Home-Bildschirm.

## Datenschutz

Es gibt keinen Server, kein Konto, kein Tracking und keine Cloud-Datenbank. Beim Löschen der Safari-Websitedaten können die lokalen Einträge verloren gehen.
